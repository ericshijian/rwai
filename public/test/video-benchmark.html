RWAI Video Benchmark

Compare YouTube, Bilibili, and native R2 playback on one page.

The page probes whether Google is reachable from the client. When the probe succeeds it auto-loads YouTube. When the probe fails it falls back to the native Cloudflare R2 video player and keeps Bilibili ready as a manual backup for regions where YouTube is unavailable.

Benchmark notes

  • All media URLs use HTTPS, so mixed-content risk should stay at zero.
  • Cross-origin iframe internals are browser-isolated. Chunk counts for YouTube and Bilibili are best-effort.
  • Open this page on mobile and desktop in each region to compare layout, autoplay restrictions, and error signals.

Embedded configuration

YouTube uses modestbranding=1&rel=0.

Bilibili uses high_quality=1&danmaku=0.

R2 uses a native <video controls preload="metadata"> player.
